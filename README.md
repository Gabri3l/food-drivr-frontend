# Food Drivr Web App

## Getting Started

You'll need to install webpack-dev-server globally.

This one liner will do the trick for you
```
npm install -g webpack-dev-server
```

### Installing
To install all of the projectâ€™s dependencies move inside the project folder then run
```
npm install
```
To run the webpack-dev-server to build and serve locally, run
```
npm run start
```
